# xintiao (薪跳) 💰

一个轻量级、美观的实时收入追踪工具，帮助你在工作期间实时查看自己的收入情况。

## ✨ 功能特性

### 核心功能
- **实时收入计算**：基于时薪和工作时长自动计算实时收入
- **流畅动画**：使用 `requestAnimationFrame` 实现平滑的数字滚动效果
- **持久化存储**：使用 `localStorage` 保存设置和进度，无需登录

### 隐身模式
- **老板键**：按 `Space` 或 `Esc` 键快速切换到假代码编辑器界面
- **咖啡休息暂停**：一键暂停计时，确保数据准确性

### 用户体验
- **暗色/亮色模式**：支持主题切换，保护眼睛
- **里程碑通知**：达到特定金额时显示庆祝提示（咖啡钱、午餐钱等）
- **Favicon 显示**：浏览器标签页图标实时显示当前收入
- **每日目标进度**：可视化显示每日目标完成情况

### 统计信息
- 工作时长显示
- 距离下班倒计时
- 每日目标进度条

## 🚀 使用方法

1. **打开应用**
   - 直接在浏览器中打开 `index.html` 文件

2. **初始设置**
   - 点击右上角的 ⚙️ 设置按钮
   - 输入你的时薪（¥/小时）
   - 设置每日目标（可选）
   - 设置上班时间和下班时间
   - 点击"保存设置"

3. **开始追踪**
   - 设置完成后，收入会自动开始计算
   - 数字会平滑滚动显示当前收入

4. **使用隐身模式**
   - 按 `Space` 或 `Esc` 键切换到假代码编辑器
   - 再次按 `Space` 或 `Esc` 键返回主界面

5. **暂停功能**
   - 点击"暂停"按钮暂停计时（如午餐时间）
   - 点击"继续"恢复计时

## 💡 使用技巧

1. **精确计时**：记得在休息时间使用暂停功能，确保数据准确
2. **每日重置**：应用会自动检测新的一天并重置数据
3. **手动重置**：如需手动重置今日数据，可在设置中点击"重置今日数据"
4. **隐私保护**：所有数据仅存储在本地浏览器，不会上传到任何服务器

## 🔒 隐私说明

- 所有数据仅存储在本地浏览器的 `localStorage` 中
- 不会收集任何个人信息
- 不会连接到任何服务器
- 完全离线可用
